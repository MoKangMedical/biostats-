# 🚀 超简单！一键自动部署指南

## 🎯 你只需要做3步，每步只需要复制粘贴！

---

## 第1步：登录服务器（1分钟）

### 使用腾讯云网页终端（最简单！）

1. **打开浏览器**，访问：
   ```
   https://console.cloud.tencent.com/lighthouse/instance
   ```

2. **找到你的服务器**，点击右侧的 **"登录"** 按钮

3. **选择 "标准登录"**

4. **等待黑色终端窗口出现**

5. ✅ 看到 `root@` 开头的提示符，说明已经登录成功！
